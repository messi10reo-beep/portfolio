機密文書QA — ローカル完結RAG 再現デモ

ローカルLLMはブラウザ上で動かせないため、このページは実機で実行した本物の入出力の「再現」です。質問を選ぶと、実際に生成された回答・根拠・応答時間をそのまま再生します。

収録について(盛っていません): 以下の回答・根拠チャンク・応答時間は、2026-07-02 に Apple M4 の MacBook 上で Ollama(qwen2.5:3b)を使い、実際に rag-chat の API を叩いて得た応答を無編集で保存したものです。外部APIへの送信はゼロ。ライブで動かす場合は GitHubのコードを clone してローカルで実行できます(README に手順)。

仕組み(30秒版)

  1. 社内文書51件(規程・安全マニュアル・型番カタログ)を分割し、端末内でベクトル化して保存。
  2. 質問もベクトル化して意味で検索。型番(YB-6208等)の取り違えは完全一致ブーストで解消 — 根拠チャンクの score 1000 がそれです。検索Recall@1 85%→100%。
  3. 上位チャンクだけを根拠にローカルLLMが回答。根拠が無い質問には「資料には記載がありません」と答える設計(ハルシネーション率 20%→5%)。
